**Admin Dashboard with Login and Roles & Permissions**
**Project Overview**
This project is a responsive Admin Dashboard built using HTML, CSS, and JavaScript.
It includes:
*Login Page with validation
*Dashboard Page with statistics cards
*Roles & Permissions management page
*Responsive sidebar navigation
*Add / Edit / Delete role functionality

**How to Run the Project**

**Method 1: Open Directly**

1.Download or clone the repository.
2.Open the project folder.
3.Double-click index.html.
4.Enter valid email and password (minimum 6 characters).
5.You will be redirected to the dashboard.

**Method 2: Using Live Server**

1.Open the project in VS Code.
2.Install the "Live Server" extension.
3.Right-click index.html.
4.Click "Open with Live Server".
5.The project will run in your browser.

**Login Validation Rules**

*Email must be in valid format.
*Password must contain minimum 6 characters.

**Instructions for running the project**

**How to Run the Project (Laptop / Desktop)**

1.Download or clone the repository.
2.Extract the ZIP file (if downloaded as ZIP).
3.Open the project folder.
4.Open the file index.html in any web browser (Chrome, Edge, Firefox).
5.The Login Page will appear.
6.Enter any valid email format and a password with minimum 6 characters to access the dashboard.
Example:
Email: admin@gmail.com
Password: 123456
After login you will be redirected to the Admin Dashboard.
